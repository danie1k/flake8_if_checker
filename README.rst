***********************
Flake8 IfChecker plugin
***********************

Warnings
--------
This plugins adds 1 new flake8 warning(s)

* :code:`IF01` Too many conditions in If/ELIF Statement/Expression.

Configuration
-------------
* :code:`max-if-conditions` By default :code:`2`.
