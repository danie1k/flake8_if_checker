Flake8 IfChecker plugin
=======================

Warnings
--------
This plugins adds 1 new flake8 warning(s)

- `IF01` Too many conditions in If/ELIF Statement/Expression.

Configuration
-------------
- `max-if-conditions` By default `2`.
